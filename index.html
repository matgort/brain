Brain Map

Organize many different threads of writing in one place. Spread ideas across an infinite canvas, connect related thoughts, and gather them into named groups.

Write

Create a note by dragging a rectangle on empty canvas. On a phone, tap New Card and then drag the note’s rectangle. Type inside the new note.

Prompt new writing with a question, fragment, quotation, or task on its own card, then build related responses around it.

Edit by clicking the text. A note keeps its drawn width and grows downward only when its writing overflows. Press Shift + Enter to finish and fit its height exactly to the writing. Linked notes grow independently.

Move or resize with a card’s edges and corners. A note can be drawn or manually resized as narrow as 40 pixels, or expanded to any larger width or height; its text wraps to the width you choose. Use Text for editing, type size, bold, italic, underline, and alignment.

Organize

Change color by opening Swatch and choosing a color. Then click a note to recolor it, or click empty canvas to recolor the background. If your text cursor is flashing inside a note, choosing a swatch recolors that note immediately.

Link lets you draw a freehand line or small picture from one note stack to another. The drawing remains attached and warps when either note moves. Draw between the same two notes again to unlink them; the note you release on receives a new random color.

Label names a connected group. Click its label card to collapse or reopen the group.

Delete removes the selected note or group after confirmation.

Navigate

The canvas is effectively infinite, so threads can spread beyond the screen without needing pages.

Pan with a trackpad or mouse wheel, or hold Space while dragging. On a phone, swipe empty canvas. Pinch, or use Command/Control + scroll, to zoom.

Double-click a card to zoom toward it. Center frames the full board. Fullscreen hides the browser chrome for more canvas space.

Keep your work

Your board autosaves in this browser. It is not synced to other browsers or devices.

Save exports the board as an image, readable text, or Code—a complete JSON backup. Email always prepares the complete Code backup and opens your phone’s share sheet so you can send it through Mail.

Import restores a Code backup. Make regular Code backups before clearing browser data.

Reset permanently wipes the current browser’s board after confirmation.
